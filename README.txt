Student Report Generator README

What is this?
---------------

A simple and efficient web application for managing student academic records. This tool allows teachers to input, store, and view student marks with automatic grade calculation and report generation.

Key Features:
- Add student records with name, roll number, and marks for three subjects
- Automatic calculation of total marks, average score, and final grade
- Display all student records in a clean, organized format
- Clear all records with confirmation prompt
- Responsive design that works on desktop and mobile devices
- Data validation to ensure accurate input

How to Use:
1. Open BI_Minor_Project.html in any modern web browser
2. Fill in the student's full name
3. Enter the roll number
4. Input marks for three subjects (each out of 100)
5. Click "Add Student" to save the record
6. Use "Display All Records" to view all entered students
7. Use "Clear Records" to remove all data (with confirmation)

Grading System:
- 90-100: A+ (Outstanding)
- 80-89: A (Excellent)
- 70-79: B (Good)
- 60-69: C (Satisfactory)
- 50-59: D (Needs Improvement)
- Below 50: F (Fail)

For Teachers/Administrators:
- No installation required - runs directly in web browser
- Simple interface that requires minimal training
- All data is stored temporarily during the session
- Easy to print or screenshot reports for record keeping

Files Included:
- BI_Minor_Project.html (complete application with HTML, CSS, and JavaScript)
- Wallpaper.jpg (background image for the application)

Technical Notes:
- Built with pure HTML, CSS, and JavaScript
- No external dependencies or libraries required
- Data persists only during the current browser session
- Fully client-side - no server or internet connection needed

Tips:
- All fields are required before adding a student
- Marks must be between 0 and 100 for each subject
- Use "Clear Records" carefully as data cannot be recovered
- Records are displayed in the order they were added

Perfect for classroom use, quick assessments, and academic record management!
